Getting Started

Welcome

Do you use multiple API's?

As ROBLOX has evolved it has developed more and more capibilities. Just to name a few: Google Sheets, Trello and Discord.
As each of these has been added, you have had to add more and more API's, making your game more and more complex. Our API will gave you the ability to reduce that number to one.

  • Only one API script.
  • Al you could ever want.
  • Helping people who don't know how to script to learn.

Setting up

How to set up our system:
1. Take the model here
2. Insert the model
3.Fill in the settings script
4. Enable HTTP Services

5. In a serverscript add the following code: 
local UDAPI = require(insert ID here)
local ServiceName1,ServiceName2 =  UDAPI:GetModule(Service1,Service2)

6. You can now use the API by doing: 
ServiceName:APIFunction(Varibles)

Features

  • Easy to use and simple
  • 100% free to use
  • Many different modules, such as Slack, Trello and Discord
  • Each module has many different commands
